# 💊 МедКонтроль

Контроль приёма лекарств для всей семьи - быстро, просто и удобно.

**🌐 [Открыть приложение](https://github.com/ваше-имя/medcontrol)**

## ✨ Возможности

- 📱 **Работает как приложение** — установи на телефон прямо из браузера
- 💊 **Управление лекарствами** — добавляй лекарства и расписание приёма
- 👥 **Семейные группы** — пригласи членов семьи по коду
- 📅 **История приёма** — отслеживай историю по дням
- 🔔 **Напоминания** — автоматические уведомления в нужное время
- 📊 **Статистика** — видь кто и когда принял лекарства
- ⚡ **Работает офлайн** — данные синхронизируются автоматически
- 🔐 **Безопасно** — все данные в зашифрованной базе Supabase

## 🚀 Быстрый старт

### На Android (Chrome):
1. Откройте сайт
2. Нажмите ⋮ (три точки) → **"Установить приложение"**
3. Приложение добавится на главный экран

### На iOS (Safari):
1. Откройте сайт
2. Нажмите **Поделиться** → **"На экран Домой"**
3. Назовите приложение

### На ПК (Chrome/Edge):
1. Откройте сайт
2. В адресной строке справа нажмите **"Установить"**
3. Приложение откроется в отдельном окне

## 🛠️ Технологии

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** Supabase (PostgreSQL + Auth)
- **PWA:** Service Worker, Web App Manifest
- **Мобильное:** Responsive Design, Touch-optimized

## 📊 Функции

### 👤 Аутентификация
- Регистрация по email
- Вход в аккаунт
- Восстановление пароля (через Supabase)

### 💊 Управление лекарствами
- Администратор добавляет лекарства
- Установка времени приёма (утро/день/вечер)
- Редактирование и удаление лекарств
- Включение/отключение напоминаний

### 👨‍👩‍👧‍👦 Управление семьей
- Создание семей
- Приглашение членов по коду (6 цифр)
- Просмотр всех участников
- Удаление участников (админом)

### 📋 Отметка приёма
- Клик на препарат = отметить что принято
- Видно кто и когда отдал лекарство
- Быстрая отмена отметки

### 📅 История и статистика
- История приёма по дням
- Выбор даты для просмотра
- Статистика соблюдения (количество принятых)

### 🔔 Уведомления
- Браузерные уведомления в нужное время
- История уведомлений в приложении
- Настройка включения/отключения

Приложение полностью соответствует стандартам PWA:

✅ HTTPS (GitHub Pages автоматически)  
✅ Service Worker (кэширование и офлайн)

✅ Responsive Design (мобильная оптимизация)  
✅ Fast Loading (оптимизированные ресурсы)
